Downloading

In order to install and use SoviotOnion, you must first download the code. To do this:
  • Download the code in a ZIP file using the green button in the top right hand corner
  • Extract the File
  • Ensure that nothing was corrupted
Afterwards, make sure that you are in the directory of the file. The following tutorial will be for Linux, as that is what I use for devlopment:
  • cd into the the file directory `cd {dir}`
  • Install all essential packages `pip install requirements.txt
  • Paste your bot token into token.txt
  • Run the bot `python main.py`
