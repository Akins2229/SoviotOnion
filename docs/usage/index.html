Usage

The following tutorial will be for Linux, as that is what I use for development.
  • Install SoviotOnion
  • Run SoviotOnion
  • Input your chosen token, which will herefprth be refered to as {your_token}
  • Use the help command for information on the amount of categories; `{your_prefix}help`
  • You may further use the help command for information on specific categories; `{your_prefix}help {category}`
  • Refer to the usage guides within the commands, and use them accordingly
Example: (With prefix ".")
.ban @Akins#2229 british
In the above example, I am banning the user "Akins#2229" for being british.
